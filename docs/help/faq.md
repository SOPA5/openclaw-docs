# 자주 묻는 질문 (FAQ)

OpenClaw 사용 중 자주 발생하는 질문들에 대한 답변입니다.

## 초기 설정 관련​ (#초기-설정-관련)

- 설치가 안 돼요:`openclaw doctor` 명령어를 실행하여 현재 환경의 문제를 진단해보세요.

- 에이전트가 응답하지 않아요: 게이트웨이가 정상적으로 실행 중인지 (`openclaw gateway status`), 그리고 모델 API 키가 유효한지 확인하세요.

## 대시보드 관련​ (#대시보드-관련)

- 대시보드 접속 오류: 기본적으로 대시보드는 로컬 호스트(`127.0.0.1:18789`)에서 작동합니다. 원격 서버인 경우 SSH 터널링이 필요합니다.

## 마이그레이션​ (#마이그레이션)

- 컴퓨터를 옮기고 싶어요:`~/.openclaw` 폴더 전체를 복사하면 새로운 기기에서도 기존 설정을 그대로 유지할 수 있습니다.

## 기타​ (#기타)

- 더 자세한 도움이 필요하다면 Discord 커뮤니티 (https://discord.gg/openclaw)에 참여하세요.
