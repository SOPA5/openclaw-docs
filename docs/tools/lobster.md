# Lobster: 규격화된 워크플로우 엔진

Lobster는 반복 작업을 **정해진 순서와 규칙대로** 밀어주는 워크플로우 엔진입니다.

## 📌 이 문서에서 배우는 것
- 언제 Lobster가 유리한지
- 일반 도구 호출과 무엇이 다른지
- 반복 작업에 왜 강한지

:::tip 쉽게 이해하기
Lobster는 **레고 조립 설명서**와 비슷합니다. 그냥 감으로 만드는 게 아니라, 1번 다음 2번처럼 순서를 정해 둬서 실수를 줄여 줍니다.
:::

## 핵심 이해
- 같은 단계를 자주 반복하는 작업에 특히 잘 맞습니다.
- 즉흥 실행보다 규격화된 순서가 필요한 상황에서 가치가 큽니다.
- 길고 복잡한 작업일수록 누락 방지에 도움이 됩니다.

## 실전 예시
- **작은 테스트 먼저**: 운영 전에 가장 작은 범위에서 먼저 확인합니다.
- **도구 선택 점검**: 정말 이 도구가 맞는지, 더 단순한 도구로 해결되는지 먼저 봅니다.
