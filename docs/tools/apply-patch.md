# apply_patch 도구

apply_patch는 파일의 **특정 부분만 정확히 바꾸고 싶을 때** 쓰는 패치형 수정 도구입니다.

## 📌 이 문서에서 배우는 것
- 언제 apply_patch가 좋은지
- 작은 수정과 큰 수정의 차이
- 왜 정확한 문맥이 중요한지

:::tip 쉽게 이해하기
apply_patch는 종이 문서 전체를 다시 쓰는 게 아니라, **빨간 펜으로 필요한 문장만 정확히 고치는 것**과 비슷합니다.
:::

## 핵심 이해
- 같은 파일의 일부만 조심스럽게 바꿀 때 유리합니다.
- 정확한 앞뒤 문맥이 있어야 패치가 잘 맞습니다.
- 복잡한 대량 수정에는 전체 편집보다 불리할 수 있습니다.

## 실전 예시
- **작은 테스트 먼저**: 운영 전에 가장 작은 범위에서 먼저 확인합니다.
- **도구 선택 점검**: 정말 이 도구가 맞는지, 더 단순한 도구로 해결되는지 먼저 봅니다.
