# 온보딩 (macOS 앱)

macOS 컴패니언 앱 설정을 위한 가이드입니다.

## 단계:​ (#단계)

- macOS 경고 승인: 보안 프롬프트를 처리합니다.

- 로컬 vs 원격: 게이트웨이를 이 Mac에서 실행할지 아니면 원격 호스트에서 실행할지 선택합니다.

- 권한: 자동화, 알림, 손쉬운 사용, 화면 기록, 마이크 등에 대한 액세스 권한을 부여합니다.

- CLI 설치: 선택적으로 npm/pnpm을 통해 글로벌 CLI를 설치합니다.

- 온보딩 채팅: 에이전트가 자신을 소개하고 다음 단계를 안내하는 전용 세션이 시작됩니다.
