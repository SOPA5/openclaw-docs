# 활용 사례 (Use Cases)

“그래서 OpenClaw로 **정확히 뭘 할 수 있는데?**”
이 질문에 답하는 문서입니다.

OpenClaw는 단순한 채팅 장난감이 아니라, **내가 쓰는 채널에서 실제 일을 해주는 시스템**입니다.

:::tip 쉬운 비유
OpenClaw는 똑똑한 비서에게 메신저로 일을 시키는 것과 비슷해요.
다만 이 비서는 말만 잘하는 게 아니라, 파일도 보고, 웹도 찾고, 정해진 일을 자동으로 처리할 수 있습니다.
:::

## 이 문서에서 하는 일

- 대표적인 활용 장면 보기
- 어떤 사람에게 어떤 쓰임이 맞는지 감 잡기
- 내 상황에 맞는 첫 사용법 떠올리기

## 먼저 알아두기

OpenClaw 소개와 설치 방법은 여기서 길게 반복하지 않겠습니다.
자세한 내용은 [설치 개요](/install/)를 참고하세요.

## 1) 내 파일을 대신 찾아주는 비서

집이나 사무실 컴퓨터에 OpenClaw가 연결되어 있으면, 밖에서도 메신저로 파일 일을 부탁할 수 있습니다.

### 이런 식으로 쓸 수 있어요
- “어제 만든 기획안 파일 찾아줘.”
- “이 문서 3줄 요약해줘.”
- “워드 파일을 PDF로 바꿔줘.”

### 왜 좋을까요?
- 파일을 일일이 클라우드에 올리지 않아도 됩니다.
- 내가 이미 갖고 있는 폴더 구조를 그대로 활용할 수 있습니다.
- 이동 중에도 급한 확인이 가능합니다.

## 2) 실시간 웹 조사 도우미

OpenClaw는 웹 검색과 브라우저 도구를 써서 최신 정보를 찾아 정리할 수 있습니다.

### 이런 식으로 쓸 수 있어요
- “오늘 애플 관련 큰 뉴스 3개만 정리해줘.”
- “경쟁사 홈페이지 바뀐 점만 알려줘.”
- “이 링크 내용 초등학생도 이해하게 요약해줘.”

### 왜 좋을까요?
- 광고와 잡음이 많은 페이지를 직접 다 뒤지지 않아도 됩니다.
- 핵심만 먼저 받아볼 수 있습니다.
- 조사 → 요약 → 전달까지 한 흐름으로 이어집니다.

## 3) 반복 작업 자동화

이 부분이 OpenClaw의 진짜 매력입니다.
질문에 답하는 것에서 끝나지 않고, **정해진 일을 대신 처리**할 수 있습니다.

### 이런 식으로 쓸 수 있어요
- 회의가 끝나면 녹음 요약 만들기
- 보고서 초안 자동 생성하기
- 정해진 시간마다 점검 메시지 보내기
- 특정 이벤트가 오면 다른 앱으로 넘기기

### 왜 좋을까요?
- 사람이 매번 눌러야 하는 일을 줄일 수 있습니다.
- “한 번 정해두면 계속 쓰는 흐름”을 만들기 좋습니다.
- 작은 팀이나 1인 업무에 특히 강합니다.

## 4) 여러 채널을 한 번에 운영하기

같은 로직을 Telegram, WhatsApp, Discord, Slack 같은 여러 채널에 연결할 수 있습니다.

### 이런 식으로 쓸 수 있어요
- 고객 문의를 여러 메신저에서 받기
- 팀 알림을 여러 채널로 보내기
- 채널은 다르지만 같은 업무 규칙을 유지하기

### 왜 좋을까요?
- 채널마다 봇을 따로 새로 만들 필요가 줄어듭니다.
- 운영 규칙을 하나의 중심 흐름으로 관리하기 쉽습니다.

## 5) 집이나 개인 환경을 위한 로컬 비서

OpenClaw를 내 장치나 내 서버에서 돌리면, 더 개인적인 일을 맡기기 좋습니다.

### 이런 식으로 쓸 수 있어요
- “서버 상태 확인해줘.”
- “오늘 백업이 잘 끝났는지 알려줘.”
- “집 장치 상태 요약해줘.”

### 왜 좋을까요?
- 중요한 데이터를 외부 서비스에 덜 맡길 수 있습니다.
- 개인 환경에 맞춘 자동화를 만들기 좋습니다.

## 6) 모바일 기기를 눈과 귀로 쓰기

iOS나 Android 기기를 node처럼 연결하면, 스마트폰 기능을 OpenClaw가 더 잘 활용할 수 있습니다.

### 이런 식으로 쓸 수 있어요
- 영수증 사진 읽기
- 현장 사진 설명 받기
- 음성으로 빠르게 지시하기

### 왜 좋을까요?
- 스마트폰이 단순한 채팅창을 넘어서 **입력 장치**가 됩니다.
- 이동 중 업무와 연결하기 좋습니다.

## 어떤 활용부터 시작하면 좋을까요?

처음에는 너무 크게 시작하지 않는 것이 좋습니다.

### 추천 시작 순서
1. Telegram 같은 채널 하나 연결하기
2. 파일 요약이나 웹 요약처럼 쉬운 작업부터 해보기
3. 자주 반복되는 일을 하나 골라 자동화해보기

이 순서가 좋은 이유는, **작은 성공 경험을 빨리 만들 수 있기 때문**입니다.

:::tip ✅ 성공하면 이렇게 보입니다
- “질문하면 답해주는 봇” 느낌에서 끝나지 않습니다.
- 내 채팅창에서 실제 파일, 링크, 업무 흐름이 연결되기 시작합니다.
- 가장 좋은 신호는 “아, 이건 매일 쓸 수 있겠다”라는 생각이 드는 것입니다.
:::

## 내 상황에 맞는 예시 한 줄

- 학생: “강의 자료 요약해줘.”
- 마케터: “경쟁사 뉴스만 모아줘.”
- 개발자: “PR 변경점만 정리해줘.”
- 운영 담당자: “오늘 에러 로그 핵심만 보여줘.”
- 1인 창업가: “고객 문의 답변 초안 만들어줘.”
