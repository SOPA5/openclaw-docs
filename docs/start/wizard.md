# 온보딩 마법사 (CLI)

온보딩 마법사는 macOS, Linux 또는 Windows(WSL2)에서 OpenClaw를 설정하는 권장 방법입니다.

```
openclaw onboard

```

## 마법사가 설정하는 항목:​

- 모델/인증: Anthropic API 키, OAuth, OpenAI 등 설정.

- 워크스페이스: 에이전트 파일 위치 설정 (기본값 `~/.openclaw/workspace`).

- 게이트웨이: 포트, 바인드 주소, 인증 모드, Tailscale 노출 여부 설정.

- 채널: WhatsApp, Telegram, Discord 등 설정.

- 데몬: LaunchAgent(macOS) 또는 systemd(Linux) 설치.

- 스킬: 권장 스킬 및 종속성 설치.
