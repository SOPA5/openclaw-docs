# 시작하기 (Getting Started)

OpenClaw를 설치하고 AI 에이전트와 첫 대화를 나누기까지의 과정을 안내합니다.

## 📋 필수 준비물​ (#-필수-준비물)

- Node.js: v22 이상 버전이 필요합니다.

- 운영체제: macOS, Linux, 또는 Windows (WSL2 권장).

```
node --version

```

---

## 🚀 빠른 시작 (Quick Setup)​ (#-빠른-시작-quick-setup)

### 1. OpenClaw 설치​ (#1-openclaw-설치)

시스템 환경에 맞는 설치 명령어를 터미널에 입력하세요.

macOS / Linux:

```
curl -fsSL https://openclaw.ai/install.sh | bash

```

Windows (PowerShell):

```
iwr -useb https://openclaw.ai/install.ps1 | iex

```

상세 설치 방법 보기 (/install)

### 2. 온보딩 마법사 실행​ (#2-온보딩-마법사-실행)

설치 후 다음 명령어를 실행하여 AI 모델(Claude, GPT 등)과 메시징 채널을 연결합니다.

```
openclaw onboard

```

온보딩 마법사 가이드 (/start/wizard)

### 3. 게이트웨이 상태 확인​ (#3-게이트웨이-상태-확인)

모든 설정이 완료되면 게이트웨이가 정상적으로 작동하는지 확인합니다.

```
openclaw gateway status

```

---

## 🖥️ 대시보드 접속​ (#️-대시보드-접속)

OpenClaw는 웹 기반의 제어판을 제공합니다. 에이전트의 작업 현황과 세션을 한눈에 볼 수 있습니다.

```
openclaw dashboard

```

브라우저에서 `http://127.0.0.1:18789/`로 바로 접속할 수도 있습니다.
