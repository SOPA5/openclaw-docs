# Matrix

Matrix는 OpenClaw를 **실제 대화창에 들여보내는 방법** 중 하나입니다. 어떤 채널이든 원리는 비슷하지만, Matrix는 특히 **자체 호스팅과 프라이버시를 중요하게 볼 때** 잘 맞습니다.

## 📌 이 문서에서 배우는 것
- 언제 Matrix를 쓰면 좋은지
- 연결 준비물은 무엇인지
- 처음 테스트를 어떻게 하면 좋은지
- 운영할 때 주의할 점

:::tip 쉽게 이해하기
채널 연결은 **새 친구를 집에 초대하는 것**과 비슷합니다. 주소와 출입문 비밀번호가 맞아야 들어올 수 있듯이, 토큰·시크릿·앱 설정이 맞아야 OpenClaw가 대화방에 들어올 수 있습니다.
:::

## 언제 쓰면 좋은가
- 자체 호스팅과 프라이버시를 중요하게 볼 때
- 조직이 이미 Matrix 서버를 쓰고 있을 때
- 개방형 프로토콜로 유연하게 연결하고 싶을 때

## 설정 개요
1. Matrix 홈서버와 봇 계정을 준비합니다.
2. 액세스 토큰과 서버 주소를 확인합니다.
3. OpenClaw에 Matrix 연결 정보를 넣습니다.
4. 테스트 룸이나 DM에서 메시지를 보내 확인합니다.

## ✅ 성공 확인
- 테스트 대화에서 메시지를 보냈을 때 응답이 돌아옵니다.
- 연결 오류 없이 채널 앱 또는 봇이 정상 상태로 보입니다.
- DM 또는 작은 테스트 방에서 기본 질의응답이 됩니다.

## 운영 팁
- 자체 서버를 쓴다면 네트워크와 SSL 상태를 먼저 점검하세요.
- 방 권한과 초대 정책을 미리 정하면 운영이 편합니다.
- 처음엔 작은 테스트 룸 하나로만 확인하세요.

## 실전 예시
- **개인 비서형**: 나만 보는 테스트 대화에서 일정 요약이나 알림을 받습니다.
- **운영형**: 작은 테스트 방에서 팀 알림, 질문 답변, 상태 확인 흐름을 먼저 검증합니다.

## 주의할 점
- 처음부터 공개 채널이나 큰 그룹에 넣지 마세요.
- 토큰·시크릿은 일반 채팅방에 올리지 마세요.
- 채널별 멘션 규칙, 답장 규칙, 알림 빈도를 먼저 정해두면 운영이 편합니다.
