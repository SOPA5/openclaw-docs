# WhatsApp

WhatsApp은 OpenClaw를 **실제 대화창에 들여보내는 방법** 중 하나입니다. 어떤 채널이든 원리는 비슷하지만, WhatsApp은 특히 **실제 일상 메신저처럼 쓰고 싶은 경우**에 잘 맞습니다.

## 📌 이 문서에서 배우는 것
- 언제 WhatsApp을 쓰면 좋은지
- 연결 준비물은 무엇인지
- 처음 테스트를 어떻게 하면 좋은지
- 운영할 때 주의할 점

:::tip 쉽게 이해하기
채널 연결은 **새 친구를 집에 초대하는 것**과 비슷합니다. 주소와 출입문 비밀번호가 맞아야 들어올 수 있듯이, 토큰·시크릿·앱 설정이 맞아야 OpenClaw가 대화방에 들어올 수 있습니다.
:::

## 언제 쓰면 좋은가
- 실제 일상 메신저처럼 쓰고 싶을 때
- 고객 응대나 알림을 자연스럽게 받고 싶을 때
- 해외 사용자와 익숙한 메신저를 쓰고 싶을 때

## 설정 개요
1. WhatsApp용 앱 또는 연동 정보를 준비합니다.
2. OpenClaw 채널 설정에 토큰·연결 값을 넣습니다.
3. 개인 DM이나 테스트 대화에서 첫 메시지를 보냅니다.
4. 답장이 오면 운영용 대화방으로 범위를 넓힙니다.

---

## 🚀 처음부터 끝까지 한 번에 — 완결 설정 흐름

### Step 1. WhatsApp 채널 로그인

터미널에서 아래 명령을 실행합니다:

```bash
openclaw channels login --channel whatsapp
```

:::tip 이 명령이 뭘 하는 건가요?
이 명령을 실행하면 OpenClaw가 WhatsApp과 연결할 준비를 해요.  
QR 코드가 터미널에 나타나는데, 폰으로 찍으면 연결이 완성됩니다!  
마치 WhatsApp Web처럼 연결하는 방식이에요. 📱
:::

---

### Step 2. QR 코드 페어링

1. 터미널에 **QR 코드**가 나타납니다.
2. 스마트폰에서 **WhatsApp 앱** 열기
3. 우측 상단 **⋮ (더보기)** → **연결된 기기** → **기기 연결**
4. QR 코드 스캐너로 터미널의 QR 코드를 찍습니다.
5. "연결 중…" 메시지 후 성공 메시지가 나타납니다.

---

### Step 3. config 파일에 설정 넣기

`openclaw.json` 파일에 아래처럼 추가합니다:

```json
{
  "channels": {
    "whatsapp": {
      "enabled": true,
      "sessionPath": "~/.openclaw/whatsapp-session",
      "dmPolicy": "knownOnly"
    }
  }
}
```

:::tip 이 설정이 뭘 하는 건가요?
- `enabled` → WhatsApp 채널을 켜는 스위치예요. `true`면 켜진 거예요.
- `sessionPath` → QR 코드로 연결한 정보를 저장하는 폴더예요. 재시작해도 다시 찍지 않아도 돼요.
- `dmPolicy` → 누가 봇한테 메시지를 보낼 수 있는지 정하는 문지기예요.
:::

---

### Step 4. Gateway 재시작

```bash
openclaw gateway restart
```

## 🔧 트러블슈팅

### QR 코드가 만료됐을 때

QR 코드는 **약 60초** 후 만료됩니다. 찍기 전에 만료되면:

```bash
# 다시 로그인 명령 실행
openclaw channels login --channel whatsapp
```

새로운 QR 코드가 나타납니다. 빠르게 찍으세요!

:::tip QR 코드가 자꾸 만료돼요
터미널 화면이 보이는 상태에서 폰을 미리 QR 스캐너 모드로 열어두고, 준비되면 명령을 실행하세요. 속도가 핵심이에요! ⚡
:::

---

### 재연결이 필요할 때

Gateway를 오래 끄거나 세션이 끊기면 다시 연결해야 합니다:

```bash
# 1. 기존 세션 파일 삭제
rm -rf ~/.openclaw/whatsapp-session

# 2. 다시 로그인
openclaw channels login --channel whatsapp

# 3. QR 코드 다시 찍기
```

:::caution 세션 파일을 지웠다면 꼭 다시 QR 찍어야 해요
세션 파일은 "이미 인증된 연결 정보"예요. 이걸 지우면 폰에서 WhatsApp도 기기 연결 해제됐다가 다시 연결돼요.
:::

## ✅ 성공 확인
- 테스트 대화에서 메시지를 보냈을 때 응답이 돌아옵니다.
- 연결 오류 없이 채널 앱 또는 봇이 정상 상태로 보입니다.
- DM 또는 작은 테스트 방에서 기본 질의응답이 됩니다.

## 운영 팁
- 처음에는 개인 테스트 대화 하나로만 검증하세요.
- 알림 메시지가 많다면 발송 빈도를 조절하세요.
- 운영용 계정과 개인 계정을 분리하면 관리가 편합니다.

## 실전 예시
- **개인 비서형**: 나만 보는 테스트 대화에서 일정 요약이나 알림을 받습니다.
- **운영형**: 작은 테스트 방에서 팀 알림, 질문 답변, 상태 확인 흐름을 먼저 검증합니다.

## 주의할 점
- 처음부터 공개 채널이나 큰 그룹에 넣지 마세요.
- 토큰·시크릿은 일반 채팅방에 올리지 마세요.
- 채널별 멘션 규칙, 답장 규칙, 알림 빈도를 먼저 정해두면 운영이 편합니다.
