# WhatsApp

WhatsApp은 OpenClaw를 **실제 대화창에 들여보내는 방법** 중 하나입니다. 어떤 채널이든 원리는 비슷하지만, WhatsApp은 특히 **실제 일상 메신저처럼 쓰고 싶은 경우**에 잘 맞습니다.

## 📌 이 문서에서 배우는 것
- 언제 WhatsApp을 쓰면 좋은지
- 연결 준비물은 무엇인지
- 처음 테스트를 어떻게 하면 좋은지
- 운영할 때 주의할 점

:::tip 쉽게 이해하기
채널 연결은 **새 친구를 집에 초대하는 것**과 비슷합니다. 주소와 출입문 비밀번호가 맞아야 들어올 수 있듯이, 토큰·시크릿·앱 설정이 맞아야 OpenClaw가 대화방에 들어올 수 있습니다.
:::

## 언제 쓰면 좋은가
- 실제 일상 메신저처럼 쓰고 싶을 때
- 고객 응대나 알림을 자연스럽게 받고 싶을 때
- 해외 사용자와 익숙한 메신저를 쓰고 싶을 때

## 설정 개요
1. WhatsApp용 앱 또는 연동 정보를 준비합니다.
2. OpenClaw 채널 설정에 토큰·연결 값을 넣습니다.
3. 개인 DM이나 테스트 대화에서 첫 메시지를 보냅니다.
4. 답장이 오면 운영용 대화방으로 범위를 넓힙니다.

## ✅ 성공 확인
- 테스트 대화에서 메시지를 보냈을 때 응답이 돌아옵니다.
- 연결 오류 없이 채널 앱 또는 봇이 정상 상태로 보입니다.
- DM 또는 작은 테스트 방에서 기본 질의응답이 됩니다.

## 운영 팁
- 처음에는 개인 테스트 대화 하나로만 검증하세요.
- 알림 메시지가 많다면 발송 빈도를 조절하세요.
- 운영용 계정과 개인 계정을 분리하면 관리가 편합니다.

## 실전 예시
- **개인 비서형**: 나만 보는 테스트 대화에서 일정 요약이나 알림을 받습니다.
- **운영형**: 작은 테스트 방에서 팀 알림, 질문 답변, 상태 확인 흐름을 먼저 검증합니다.

## 주의할 점
- 처음부터 공개 채널이나 큰 그룹에 넣지 마세요.
- 토큰·시크릿은 일반 채팅방에 올리지 마세요.
- 채널별 멘션 규칙, 답장 규칙, 알림 빈도를 먼저 정해두면 운영이 편합니다.
