# 메시징 채널 (Channels)

OpenClaw는 여러분이 평소 사용하는 메시징 앱을 통해 에이전트와 대화할 수 있게 해줍니다. 한 번의 설정으로 여러 채널을 동시에 운영할 수 있습니다.

## 🚀 기본 설정 흐름​

대부분의 채널은 다음과 같은 공통 과정을 거칩니다.

- 플랫폼 개발자 도구 등록: 각 서비스(Discord Developer Portal 등)에서 봇이나 앱을 생성합니다.

- 인증 정보 확보: API Key, Token, 또는 Secret 등을 복사합니다.

- OpenClaw와 연결: `openclaw configure` 명령어나 대시보드를 통해 정보를 입력합니다.

- 페어링(Pairing): 안전한 통신을 위해 초기 페어링 과정을 거칩니다.
