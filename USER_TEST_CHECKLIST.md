# OpenClaw Docs User Test Checklist

## 목적
처음 방문한 사용자가 길을 잃지 않고 설치/검색/다음 단계로 넘어가는지 확인하기 위한 체크리스트.

## 테스트 대상
1. 완전 초보자
2. 이미 설치한 사용자
3. 환경이 헷갈리는 사용자 (WSL2 vs Windows Native)

## 시나리오
### A. 처음 방문
- [ ] 홈페이지에서 10초 안에 "무엇을 해야 하는지" 이해한다
- [ ] 빠른 시작 3단계를 찾을 수 있다
- [ ] 내 환경에 맞는 설치 링크를 찾을 수 있다

### B. 검색
- [ ] 검색창 존재를 바로 인지한다
- [ ] Ctrl/⌘ + K 단축키를 이해한다
- [ ] "WSL2", "Telegram", "subagent" 검색 시 원하는 문서를 찾을 수 있다

### C. 문서 흐름
- [ ] 현재 문서 단계(Beginner/Core/Advanced)를 이해한다
- [ ] 다음 읽을 문서를 따라간다
- [ ] 비교형 카드/경로 카드가 실제로 도움이 된다

### D. 설치/첫 실행
- [ ] 설치 개요 -> 시작하기 -> 설정 마법사 흐름이 자연스럽다
- [ ] 첫 질문 보내기까지 경로가 끊기지 않는다

## 기록 항목
- 가장 헷갈린 부분
- 클릭했지만 기대와 달랐던 부분
- 검색으로 못 찾은 키워드
- 다음 단계 링크가 부족했던 문서
- 추가되면 좋을 비교/카드
