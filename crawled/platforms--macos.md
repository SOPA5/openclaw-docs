# macOS 가이드

OpenClaw는 macOS 환경에서 가장 강력하고 직관적인 경험을 제공합니다. Apple Silicon(M1/M2/M3)과 Intel Mac을 모두 네이티브로 지원합니다.

## 🚀 설치 방법​ (#-설치-방법)

### 1. DMG 파일 사용​ (#1-dmg-파일-사용)

공식 사이트에서 `.dmg` 파일을 내려받아 `Applications` 폴더로 드래그합니다.

### 2. Homebrew 사용​ (#2-homebrew-사용)

```
brew install --cask openclaw

```

---

## 🔒 필수 권한 설정​ (#-필수-권한-설정)

에이전트가 여러분의 맥을 대신 조작하려면 다음 권한이 필요합니다.

- 전체 디스크 접근 권한(Full Disk Access): 파일을 읽고 쓰기 위해 필요합니다.

- 손쉬운 사용(Accessibility): 마우스 클릭 및 키보드 입력을 위해 필요합니다.

- 화면 기록(Screen Recording): 브라우저나 화면의 상태를 에이전트가 '보기' 위해 필요합니다.

상세 권한 설정 방법 (/platforms/mac-permissions)

## 📂 로그 위치​ (#-로그-위치)

문제가 발생했을 때 다음 경로에서 로그를 확인할 수 있습니다.

- `~/Library/Logs/OpenClaw/`

- 로그 관리 가이드 (/platforms/mac-logging)
