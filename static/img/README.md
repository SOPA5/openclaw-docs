# static/img

이 폴더에 OpenClaw 관련 이미지를 배치합니다.

## 필요한 파일들

| 파일명 | 용도 | 출처 |
|--------|------|------|
| `logo.svg` | 내비게이션 바 로고 (현재 placeholder) | https://openclaw-wheat.vercel.app 에서 다운로드 |
| `favicon.ico` | 브라우저 탭 아이콘 | 로고에서 변환 |
| `openclaw-social-card.png` | OG/트위터 소셜 카드 (1200×630) | 직접 제작 |

## 로고 교체 방법

원본 사이트에서 실제 OpenClaw 로고를 가져와 `logo.svg`를 교체하세요.
